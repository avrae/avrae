Overview

So, what can Avrae do? Currently, some features include:

  • Advanced dice rolling
  • Character sheet integration
  • Spell and monster lookup
  • Initiative tracking

  • Efficient

  • Fast

  • Intuitive

  • Sleek

  • Practical

Dice Roller

The dice roller includes support for advanced dice syntax, allowing a user to reroll certain dice, keep only the highest few dice, and perform complex calculations.
 
 

Character Sheet Integration

Avrae can read character sheets from dicecloud or a PDF, automatically generating macros to roll attacks, ability checks, and saving throws. A player can then simply use a command to make a check, save, or attack with a weapon, and all necessary will be resolved automatically.

Spell & Monster Lookup

Avrae can look up spells, monsters, and status effects from the 5e SRD. There is a configurable option to show a generalization to players when looking up a monster.
 

Initiative Tracking

The initiative tracker is a fast way to track combat in a text channel. It supports automatic combatant sorting, HP and status effect tracking, and integration with the character sheet manager and 5e SRD to further streamline combat.
