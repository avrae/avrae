Character Sheet Integration. You'll never have to memorize a modifier again.

Avrae can read character sheets from dicecloud or a PDF, automatically generating macros to roll attacks, ability checks, and saving throws. A player can then simply use a command to make a check, save, or attack with a weapon, and all necessary rolls will be resolved automatically.

Initiative Tracking. How long left on my spell? How injured is the dragon?

The initiative tracker is a fast way to track combat in a text channel. It supports automatic combatant sorting, HP, AC, resistance, and status effect tracking, and integration with the character sheet manager and 5e SRD to further streamline combat.
